# 🔐 Encrypted Chat App (AES + GUI)

A secure multi-client chat application using AES-256 encryption and TCP sockets with a modern style GUI.

---

## 🚀 Features

- 🔐 AES-256 GCM Encryption
- 💬 Real-time messaging
- 👥 Multi-client support
- 🖥️ Professional GUI (WhatsApp style)
- 🔑 Secure nonce (IV) handling
- 📝 Encrypted logging

---

## 🛠️ Tech Stack

- Python 3
- socket programming
- threading
- Tkinter GUI
- cryptography library

---

## ⚙️ Installation

```bash
pip install cryptography
```

### ▶️ How to Run
- Step 1: Start Server
```Bash
python server.py
```
- Step 2: Run Client
```Bash
python client_gui_pro.py
```

### 🔐 Encryption Flow
Message typed by user
Encrypted using AES-256 GCM
Sent via TCP socket
Received & decrypted by other clients
