# Encrypted-chat-app


## ⚙️ Installation
```Bash
pip install cryptography
```

## ▶️ How to Run
- Step 1: Start Server
```Bash
python server.py
```
- Step 2: Start Multiple Clients (2+ terminals)
```Bash
python client.py
```

## 🔐 Security Features Included
- ✅ AES-256 GCM encryption
- ✅ Random nonce (IV) per message
- ✅ Encrypted message logging
- ✅ Multi-client support (threading)
- ✅ No plaintext transmission
